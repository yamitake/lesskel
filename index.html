Lesskel

lesskel is a library that output from the html source code skeletons for the less.

View project onGitHub

What is Lesskel?

レスケルはブラウザで表示されているHtmlソースコードから、lessのスケルトンを作成できるツールです。

How to Use.

Lesskelはブックマークレットとして機能を提供しています。ブラウザで開いている状態でブックマークレットを使用する事で、Lesskelが使用できます。 ※一部webページには対応していません。
+ lesskel ← Add this link to your Bookmarks Bar

jquery.lesskel.js

このライブラリはjquery pluginとして作られており、下記の様に使用できます。 

var elem = document.getElementById('id');//or $("#id");  ← Jquery
var src = $.lesskel(elem); // src is String. ex) div#id{}

//other
var options = {includeCssText:true}; //htmlにstyle属性が直書きしてる場合に出力する。
$.lesskel(elem , options);
